  # Abysswalker for Zed

  **A minimalist dark theme. Designed for deep focus.**
</div>

---

A port of the [Abysswalker VS Code theme](https://github.com/stkossman/abysswalker-theme) for the [Zed editor](https://zed.dev).

Tuned for **Frontend Development** (React, TypeScript, Astro, CSS).

## Palette

| Role | Hex |
|---|---|
| Background | `#1a1a1a` |
| Foreground | `#a0a0a0` |
| Keywords | `#bd7e7e` |
| Strings | `#d1c29b` |
| Functions | `#8fa8bd` |
| Types / Tags | `#f5f5f5` |
| Comments | `#606060` |
| Punctuation | `#757575` |

## Installation

Search **Abysswalker** in `zed: extensions` or add to your `~/.config/zed/settings.json`:
```json
"theme": "Abysswalker"
```

## Recommended Settings
```json
{
  "buffer_font_family": "Geist Mono",
  "buffer_font_size": 15,
  "theme": "Abysswalker"
}
```
